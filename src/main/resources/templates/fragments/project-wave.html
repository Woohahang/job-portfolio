Project Wave

Node.js 기반의 Discord 봇으로, 사용자가 게임 정보를 등록하면 음성 채널 입장 시 자동으로 표시되며,

티어 순위도 별도로 조회할 수 있습니다.

  • Riot Games API를 통한 티어 실시간 조회
  • MongoDB 기반 사용자/서버 정보 저장 및 관리
  • Slash Command 기반 인터랙션 구성
  • 사용자 게임 닉네임 등록 시 실시간 유효성 검사 및 중복 방지
  • 게임 종류에 따라 자동 포맷 및 외부 API를 통한 데이터 연동
  • 닉네임 등록 결과에 따른 상태 메시지 피드백 제공
